scrabble by Kedrigern

Classic desk game Scrabble with suport for AI (GADDAG + greedy algorithm) and multiplayer. Graphic toolkit is GTK#, language is C# (Mono). Builds are playable, but not "release quality". I hope that in January 2012 will come "true release".

Documentation:

  • Doc for users
  • Doc for programmers
  • API doc

Dependencies

  • .NET framework or Mono
  • GTK#

Additional information about install dependecies at project wiki.

Install

There is deb packge (download section).

You can run binary file without instalation. If you have all dependencies then you just run scrabble.exe (include Linux and Mac).
